FINANCIAL TRANSACTION CODE
Vocabulary Type Concept Domain
Concept Domain Financial Transaction Code
Code System Name
Code System Identifier
Code System OID
HL7 Table Identifier HL70096
Version
Status Active
Where Used
Purpose and Use HL7-defined table of concepts which, for an order, indicate the amount of information to be returned from the filler for the response to the order. Sometimes the requested level of response may not be possible immediately, but when it is possible, the filler (receiving) application must send the information. Used in HL7 Version 2.x messaging in the ORC segment.
