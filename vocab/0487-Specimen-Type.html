Specimen Type
Vocabulary Type Code System - Universal
Concept Domain Specimen Type
Code System Name
Code System Identifier
Code System OID
HL7 Table Identifier HL70487
Version
Status Active
Where Used SPM-4
Purpose and Use HL7-defined table of codes that describe the precise nature of an entity that may be used as the source material for an observation. This is one of two code systems that are used instead of table 0070 (code system 2.16.840.1.113883.18.28) which conflated specimen types and specimen collection methods.
